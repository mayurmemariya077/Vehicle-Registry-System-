## 🏗️ Inheritance Structure

```text
Vehicle
│
└── Car
    │
    ├── ElectricCar
    │      │
    │      └── SportsCar
    │
    ├── Sedan
    │
    └── SUV

Aircraft
   │
   └── FlyingCar
       ▲
       │
      Car
```

## 🚘 Supported Vehicle Types

### 🚗 Car

* Vehicle ID
* Manufacturer
* Model
* Year
* Fuel Type

### ⚡ Electric Car

* All Car Features
* Battery Capacity (kWh)

### 🏎 Sports Car

* All Electric Car Features
* Top Speed (km/h)

### ✈️ Flying Car

* All Car Features
* Flight Range (km)

### 🚘 Sedan

* Passenger Vehicle Category

### 🚙 SUV

## 🚀 Getting Started

### 📥 Clone Repository

```bash
git clone <repository-url>
```

### ⚙️ Compile

```bash
g++ VehicleRegistry.cpp -o VehicleRegistry
```

### ▶️ Run

```bash
./VehicleRegistry
```
